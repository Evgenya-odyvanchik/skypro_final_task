Командная строка Windows

Управление файлами и папками

  • COPY – Копирование файлов в другое место
  • DIR на Отображение файлов и папок в текущем каталоге
  • CD - Смена текущего каталога (папки)
  • MKDIR - создание папки
  • MOVE — перемещение файлов из одной папки в другую

Приложения и процессы

  • TASKLIST — список выполняемых задач (процессов)
  • TASKKILL — остановка выполнения задачи (для указания задачи используется PID из TASKLIST)

Системная информация

  • DATE — вывод или установка текущей даты
  • TIME — вывод или установка текущего времени
  • HOSTNAME — отображение имени компьютера
  • SYSTEMINFO — отображение подробной информации о конфигурации компьютера
  • VER — просмотр версии операционной системы Windows
